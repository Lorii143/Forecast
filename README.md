# Time Series Forecast Analysis - Road Condition Survey: EDA, Reliability, Regression & VAR

## Overview

This project focuses on analyzing historical data and building a forecasting model to predict future trends using time series techniques.

## Objective

To identify patterns in historical data and generate predictions that can support data-driven decision-making.



## Features

* Data preprocessing and cleaning
* Trend analysis
* Visualization of historical data
* Forecasting future values

## Technologies Used

* Python
* Pandas
* Matplotlib / Seaborn 

## Key Insights

* Identified trends over time
* Observed seasonal patterns 
* Generated predictions for future periods

## How to Run

1. Load the dataset
2. Run the analysis script
3. View visualizations and forecast results
